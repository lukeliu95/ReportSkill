# Notes: 钢铁行业 TOP40 分析

## 日本钢铁行业 2025-2026 趋势

### 电弧炉（EAF）转型加速
- 新日铁承诺投入**6.05亿美元**用于三家工厂的电弧炉技术改造
- 预计到2029年，每年将新增**2.9万公吨**钢铁产能
- 获得**1.75亿美元**政府补贴支持绿色转型计划

### 高炉产能持续缩减
- 到2026年，计划还将关闭**840万吨/年**的炼铁产能
- 在脱碳大趋势下，日本将继续压减钢铁产能

### 脱碳技术路径
1. **电弧炉技术**：发展全废钢电炉流程，作为碳减排的有力措施
2. **天然气还原铁制造**（MIDREX®工艺）：可减少高炉工序约**20%**的二氧化碳排放量
3. **氢能技术布局**：加快低碳技术研发

### 2025年关键节点
- 2025年底前，钢铁行业面临严格的环保和双碳目标约束
- 钢铁行业被纳入全国碳排放权交易体系

### 主要企业数据

#### 日本制铁（Nippon Steel）
- 全球粗钢产能：每年**8,600万吨**（其中4,200万吨来自海外）
- 2025-2026财年上半年钢铁产品销量：**1,547万吨**，同比下降2.8%
- 目标：到**2030年全球粗钢产能突破1亿吨**
- 2024财年营收：86,955亿日元（约603亿美元）
- 2024年归母净利润：3,502亿日元（约169亿元人民币）
- 2025年世界500强排名：第244位

#### JFE钢铁
- 2024-25财年粗钢产量：**2,320万吨**，同比下降6.4%
- 全球排名：第14位（23.53百万吨）
- 2025财年上半年实现净利润扭亏
- 2025年12月宣布以2,700亿日元收购印度JSW钢铁子公司50%股份

### 市场份额
- 日本国内：日本制铁、JFE和神户制钢三家企业占日本粗钢产量约**86%**
- 全球高端特殊钢材市场：日本占了**33%**的市场份额
